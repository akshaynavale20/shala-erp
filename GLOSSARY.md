## Notes on Key Term Choices

1. **"हजेरी" vs "उपस्थिती":** "हजेरी" is the word teachers and students actually use. "उपस्थिती" appears in formal GRs. Use "हजेरी" in the UI; use "उपस्थिती" only on formal certificates.

2. **"शाखा" dual meaning:** In Jr College context = stream (Science/Commerce/Arts). In Sanstha context = unit/branch. The UI must disambiguate — use "विभाग/घटक" for Sanstha unit and "शाखा" only for academic stream.

3. **"ज्युनियर कॉलेज" vs "कनिष्ठ महाविद्यालय":** Both correct. "ज्युनियर कॉलेज" is used in conversation and on signboards. "कनिष्ठ महाविद्यालय" appears on official certificates. Use "कनिष्ठ महाविद्यालय" on PDFs/certificates; "ज्युनियर कॉलेज" acceptable in UI navigation.

4. **"वर्तणूक दाखला" vs "बोनाफाईड":** Maharashtra schools use "वर्तणूक दाखला" for character/conduct certificate. "बोनाफाईड" (bonafide) is used for proof-of-study certificate. These are two different documents. Confirm with school which they need.

5. **"घटक चाचणी" vs "एकक चाचणी":** Maharashtra's official curriculum documents use "घटक चाचणी". Use this.

6. **"सहामाही परीक्षा":** Some schools say "अर्धवार्षिक परीक्षा". "सहामाही" is more common in practice. Confirm.

7. **Gender inflection:** Marathi inflects nouns and verbs for gender. UI strings must handle this for student-facing text (विद्यार्थी / विद्यार्थिनी). Use `i18next` gender context.
