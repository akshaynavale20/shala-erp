👨‍🎓 विद्यार्थी व्यवस्थापन — विद्यार्थी यादी

नवीन विद्यार्थी कसा जोडावा?

  1. वरच्या उजव्या कोपऱ्यातील + नवीन विद्यार्थी बटणावर क्लिक करा
  2. उघडलेल्या फॉर्ममध्ये विद्यार्थ्याचे मराठी नाव आणि इंग्रजी नाव भरा
  3. GR क्रमांक, जन्मतारीख, वडिलांचे नाव टाका
  4. इयत्ता, तुकडी, प्रवर्ग निवडा
  5. फोटो अपलोड करण्यासाठी विद्यार्थ्याच्या फोटो जागेवर क्लिक करा
  6. जतन करा बटणावर क्लिक करा

नवीन कर्मचारी कसा जोडावा?

  1. + नवीन कर्मचारी बटणावर क्लिक करा
  2. मराठी नाव, इंग्रजी नाव भरा
  3. पद (मुख्याध्यापक / शिक्षक इ.), कर्मचारी प्रकार निवडा

📅 प्रकरण ५हजेरी — दैनिक उपस्थिती

📅 दैनिक हजेरी नोंद पृष्ठ

आजची हजेरी कशी घ्यावी?

  1. डाव्या मेनूतून हजेरी वर क्लिक करा
  2. वरील ड्रॉपडाउनमधून शाळा निवडा (आपोआप निवड होते)
  3. तारीख योग्य आहे का पाहा — आज तारीख आपोआप असते
  4. प्रत्येक विद्यार्थ्यासाठी उपस्थित, अनुपस्थित, किंवा उशिरा / अर्धदिवस निवडा
  5. सर्वांना एकत्र सर्व उपस्थित किंवा सर्व अनुपस्थित बटणाने मार्क करता येते
  6. शेवटी हजेरी जतन करा बटणावर क्लिक करा

📝 प्रकरण ६परीक्षा व गुण नोंद

📝 परीक्षा व्यवस्थापन — परीक्षा यादी

नवीन परीक्षा कशी तयार करावी?

  1. + नवीन परीक्षा बटणावर क्लिक करा
  2. परीक्षेचे नाव भरा (उदा. प्रथम सत्र परीक्षा)
  3. प्रकार निवडा (अर्धवार्षिक / वार्षिक / सातत्यपूर्ण)
  4. सुरुवात व शेवट तारीख निवडा
  5. जतन करा दाबा

गुण कसे नोंदवावे?

  1. परीक्षा यादीत गुण नोंद बटणावर क्लिक करा
  2. प्रत्येक विद्यार्थ्यासाठी विषयनिहाय गुण टाका
  3. सर्व गुण टाकल्यावर जतन करा बटण दाबा
✅
गुण नोंद केल्यावर अहवाल केंद्रातून प्रगती अहवाल PDF स्वरूपात काढता येतो.

💰 प्रकरण ७शुल्क व्यवस्थापन

💰 शुल्क व्यवस्थापन — फी संकलन पृष्ठ

शुल्क संकलन (फी घेणे) कसे करावे?

  1. डाव्या मेनूतून शुल्क वर क्लिक करा
  2. शुल्क संकलन टॅब निवडला आहे का ते पाहा
  3. डाव्या पॅनलमध्ये इयत्ता आणि तुकडी निवडा, मग विद्यार्थ्याचे नाव निवडा
  4. उजव्या बाजूला विद्यार्थ्याची थकबाकी व मागणी यादी दिसेल
  5. किती फी घ्यायची ते निवडा (सर्व चेक केलेले असतात)
  6. फी घ्या बटणावर क्लिक करा
  7. भरणा पद्धत (रोख / UPI / ऑनलाइन) निवडा आणि पुष्टी करा
  8. पावती आपोआप तयार होईल — PDF डाउनलोड करा

शुल्क रचना (Fee Structure) कशी तयार करावी?

  1. शुल्क रचना टॅबवर क्लिक करा
  2. + नवीन रचना बटणावर क्लिक करा
  3. शुल्काचे नाव, रक्कम, लागू इयत्ता निवडा
  4. जतन करा दाबा — या रचनेनुसार आपोआप मागणी तयार होईल

💵 वेतन व्यवस्थापन — वेतन स्लिप यादी

नवीन वेतन स्लिप कशी तयार करावी?

  1. + नवीन वेतन स्लिप बटणावर क्लिक करा
  2. कर्मचारी निवडा — मागील महिन्याचे तपशील आपोआप भरतात
  3. महिना आणि वर्ष निवडा
  4. वेतन घटक (मूळ वेतन, DA, HRA इ.) तपासा व आवश्यक असल्यास बदला
  5. कपात (PF, PT इ.) तपासा
  6. स्लिप तयार करा बटण दाबा
  7. वेतन दिल्यावर वेतन दिले बटणावर क्लिक करा — जमा-खर्चात नोंद होईल

वेतन घटक (Salary Components) कसे सेट करावे?

  1. वेतन घटक टॅबवर क्लिक करा
  2. + नवीन घटक बटण दाबा
  3. घटकाचे नाव (Basic Pay, DA, HRA इ.) व प्रकार (उत्पन्न/कपात) निवडा
  4. जतन करा दाबा

📒 जमा-खर्च व्यवस्थापन — उत्पन्न व खर्च नोंदी

जमा (उत्पन्न) नोंद कशी करावी?

  1. + जमा नोंद बटणावर क्लिक करा
  2. विभाग (Category) निवडा — देणगी / अनुदान / इतर उत्पन्न
  3. वर्णन (मराठीत) भरा
  4. रक्कम, तारीख, भरणा पद्धत भरा
  5. जतन करा दाबा

खर्च नोंद कशी करावी?

  1. + खर्च नोंद बटणावर क्लिक करा
  2. खर्चाचा विभाग, वर्णन, रक्कम, तारीख भरा
  3. पक्ष (Party name) असल्यास भरा
  4. जतन करा दाबा

🏫 इयत्ता व तुकड्या व्यवस्थापन

नवीन इयत्ता कशी जोडावी?

  1. शाळा व शैक्षणिक वर्ष निवडा
  2. + इयत्ता जोडा बटणावर क्लिक करा
  3. इयत्तेचे नाव (उदा. इयत्ता पहिली), स्तर निवडा
  4. जतन करा दाबा

तुकडी (Division) कशी जोडावी?

  1. इयत्तेच्या ओळीत + (plus) चिन्हावर क्लिक करा
  2. तुकडीचे नाव टाका (उदा. अ, ब, क किंवा A, B)
  3. जतन करा दाबा
📌
महत्त्वाचे: नवीन शैक्षणिक वर्षाच्या सुरुवातीला इयत्ता व तुकड्या आधी जोडा — नंतरच विद्यार्थी प्रवेश करता येतो.

🏫 शाळा / घटकसंस्थेच्या अंतर्गत शाळा जोडा, शाळेचे नाव, UDISE, पत्ता
📅 शैक्षणिक वर्षेशैक्षणिक वर्ष जोडा, चालू वर्ष सेट करा
🔐 भूमिका / परवानग्यावापरकर्त्यांसाठी भूमिका तयार करा (संचालक / मुख्याध्यापक / लिपिक इ.)
👤 वापरकर्तेनवीन वापरकर्ता तयार करा, भूमिका द्या, पासवर्ड रीसेट करा

नवीन वापरकर्ता कसा तयार करावा?

  1. सेटअप → वापरकर्ते वर क्लिक करा
  2. + नवीन वापरकर्ता बटण दाबा
  3. नाव, ईमेल, भूमिका भरा
  4. प्रारंभिक पासवर्ड सेट करा
  5. जतन करा दाबा

लोगो कसा अपलोड करावा?

  1. सेटअप → संस्था माहिती वर क्लिक करा
  2. लोगो अपलोड करा बटणावर क्लिक करा
  3. PNG / JPEG / SVG फाइल निवडा (जास्तीत जास्त २ MB)
  4. जतन करा दाबा — सर्व PDF मध्ये लोगो येईल
🔒
सुरक्षा: भूमिका काळजीपूर्वक द्या. संचालकाला सर्व अधिकार असतात; इतर भूमिकांना मर्यादित प्रवेश असतो.

दैनंदिन कामे — क्रमवारी

  1. सकाळी: हजेरी → विद्यार्थी उपस्थिती नोंदवा → जतन करा
  2. फी आल्यावर: शुल्क → विद्यार्थी निवडा → फी घ्या → पावती द्या
  3. महिना अखेर: वेतन → नवीन स्लिप → तपासा → दिले मार्क करा
  4. परीक्षेनंतर: परीक्षा → गुण नोंद → जतन करा
  5. अहवाल हवा असल्यास: अहवाल केंद्र → प्रकार निवडा → मुद्रित करा

सामान्य प्रश्नोत्तरे (FAQ)

❓
Q: विद्यार्थी कसा शोधायचा?
A: विद्यार्थी पृष्ठावर शोध चौकटीत नाव किंवा GR क्रमांक टाका.
❓
Q: चुकीची हजेरी सुधारता येईल का?
A: हो — हजेरी पृष्ठावर जा, तारीख बदला, सुधारा, पुन्हा जतन करा.
❓
Q: PDF मध्ये शाळेचा लोगो येत नाही?
A: सेटअप → संस्था माहिती → लोगो अपलोड करा → जतन करा.
❓
Q: इयत्ता / तुकडी दिसत नाही?
A: सेटअप → इयत्ता व तुकड्या → योग्य शाळा व वर्ष निवडा → इयत्ता जोडा.
❓
Q: वेगळ्या शाळेचा डेटा कसा पाहायचा?
A: वरच्या हेडरमध्ये शाळा निवड ड्रॉपडाउनमधून शाळा बदला.
